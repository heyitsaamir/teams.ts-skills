# teams.ts-skills

Agent skills for building Microsoft Teams bots with the Teams SDK.

## Skills

### teams-sdk-project

Scaffolds a complete Teams SDK bot project in TypeScript — from dev tunnel setup through bot registration to a running bot.

**What it does:**
- Creates a dev tunnel (via `devtunnel`)
- Registers a bot with Azure AD and Teams Developer Portal (via `teams2` CLI)
- Scaffolds the project code (echo or AI template)
- Installs dependencies

**Install:**
```bash
npx skills add heyitsaamir/teams.ts-skills
```
